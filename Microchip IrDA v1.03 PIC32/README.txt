1. Description
---------------------------------------------------------------------------------------
This is the IrDA Library for the PIC32. The library is the same as the 16 bit library, 
but adds support for 32 bit processors. Please refer to AN01071.pdf for information on 
how the stack works, the demos available, and how to run the demos.  

---------------------------------------------------------------------------------------
2. Device Support
---------------------------------------------------------------------------------------
PIC32MX460F512L PIM
 Supports UART1 and UART2
PIC32MX795F512L PIM
 Supports UART1, UART2 and UART3
